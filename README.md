#  随机蒙题器

### 简介

你的寒假作业写完了吗？如果你想要蒙完一本寒假作业，又觉得自己蒙太有规律，就使用本仓库的软件。

### 使用
1. 使用python运行main.py,或下载exe发行版

2. 输入要蒙的个数，点击生成或按下<kbd>Enter</kbd>生成蒙题

3. 上方TextBrowser里显示生成蒙出的结果

4. 愉快地蒙题
